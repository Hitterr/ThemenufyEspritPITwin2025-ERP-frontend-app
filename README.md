# the-menufy-back-office

## How to run

1. Clone this repository

```shell
git clone https://github.com/Hitterr/the-menufy-backoffice-20.git
```

2. Run

```shell
npm i -f
```

3. Run

```shell
npm run dev
```

## Notes

- Don't forget to run the backend first
